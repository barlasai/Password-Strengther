# Password Strength Background

A simple web application that visually demonstrates password strength by dynamically blurring/unblurring a background image based on the length of the entered password.

## Features

- Real-time visual feedback on password strength
- Background image blur effect that decreases as password length increases
- Clean, responsive UI using Tailwind CSS
- Minimalist design with email and password input fields

## How to Use

1. Open `index.html` in a web browser
2. Enter an email address (optional)
3. Type in the password field and watch the background image become clearer as the password gets longer

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS framework)
- JavaScript (ES6)

## Project Structure

- `index.html` - Main HTML file
- `style.css` - Custom CSS styles
- `script.js` - JavaScript functionality for password strength visualization

## Demo

As you type in the password field, the background image will transition from heavily blurred to clear, providing immediate visual feedback on password strength.
